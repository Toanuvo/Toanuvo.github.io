|---------HomePage---------|
Overview

The home page is the first page you'll see when using this app, it displays the top 5 tasks based on the current sort setting. The home page will also greet you with your desired username and allows you to complete tasks. This page can also be sorted by a-z, z-a, 1-100, 100-1, and date.


Task Completion

To complete tasks on the home page a complete task button will be displayed next to the task. Pressing this button will complete the task and remove it from the tasks list awarding points and updating the streak counter if required.

|---------taskPage---------|
Task Completion

To complete task on the task page all that is needed is to press the desired tasks complete task button which will remove the task from the tasks and homepage updating the points and streak counter if required


Task Deletion

To Delete a task first click edit task on the task you wish to delete this will display it in the task editor. Then click the delete task button on the task editor this will remove the task from both the homepage and the tasks page


Edit Task

To edit a task simply press the edit task button on the desired task at which point the task will appear in the task editor. In this section every elements of the selected task will appear and can be changed. Once satisfied with the changed click the reconfigure button and the task will be reinserted into the task list


Clear all inputs

the clear all inputs button removes all input data from the task editor

Sort tasks

Tasks can be sorted by a-z, z-a, 1-100, 100-1, and date

A-Z: sort tasks by their name LEXOGRAPHICALLY A first

Z-A: sort tasks by their name LEXOGRAPHICALLY Z first

1-100: sort tasks by their priority lower first

100-1: sort tasks by their priority higher first

1/11/1111: sort tasks by their date sooner first

Load Data

This will allow you to load local saved data to the app so you can pick up from where you left off on a todolist it will only work if you have applied the previous username that you used


Save Data

This saves the state of the app so that you can come back to it later using your username as a way to retrieve it again

Change Font

Reset Points

Reset Streak

Home page top X results
