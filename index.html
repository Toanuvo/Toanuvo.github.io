Hello and welcome to the todo list
this page is designated for guides on the features of this app

|---------HomePage---------|
Overview

The home page is the first page you'll see when using this app, it displays the top 5 tasks by default but can be edited in the settings to 10 and 15. The home page will also greet you with your desired username and you can complete tasks. this page can also be sorted by a-z, z-a, 1-100, 100-1, and date.


Task Completion

to complete tasks on the home page a complete task button similar to the task page will appear next to the task. Pressing this button will complete the task and remove it from the tasks list awarding points and updates the streak counter if required.

|---------taskPage---------|
Task Completion

To complete task on the task page all that is needed is to press the desired tasks complete task button which will remove the task from the tasks and homepage updating the points and streak counter if required


Task Deletion

To Delete a task all that is needed is to press the delete task button which will remove the task from both the homepage and the tasks page


Edit Task

To edit a task simply press the edit task button on the desired task at which point the task will appear at the top in the elements portion of the task page. in this section every elements of the selected task will appear and can be changed. Once statisfied with the changed click the reconfigure button and the task will be reinserted into the task list


Clear all inputs

the clear all inputs button removes all input data for adding or editing a task

|---------settingsPage---------|

Load Data

this will allow you to load local saved data to the app so you can pick up from where you left off on a todolist it will only work if you have applied the previous username that you used


Save Data

this saves the state of the app so that you can come back to it later using your username as a way to retrieve it again


apply all settings
this button applys all changes in the below settings if you want to apply them all at once
Change/Update Username
This box on the settings page enables you to change the user name you would like which will be displayed on the home page after hitting the apply button
Change UI Color
This dropdown menu changes the Ui Background color from a limited color pallet it will not allow the color to be the same as the font color
Change Font Color
This dropdown menu changes the font color from a limited color pallet, it will not allow the color to be the same as the UI color
Change Font
This dropdown menu changes the font for the app
Reset Points
This button lets you reset your points if you desire.
Reset Streak
This button lets you reset your streak if you desire
home page top X results
this will change the amount of tasks shown on the homepage
